    INSTALACION DL PROGRAMA DE CONTROL ESCOLAR

    CALIFICACIONES

    Con el fin de simplificar y distribuir el trabajo que representa la nueva obligación que tienen las escuelas de educación básica (preescolar, primaria y secundaria), de informar a los padres de familia del desempeño escolar de sus hijos, asi como de las observaciones y los apoyos requeridos para mejorar su desempeño, El sistema de control escolar permite generar un sistema para los maestros, el cual contiene la información de los alumnos de cada grupo que el maestro atiende, en este sistema el maestro puede registrar calificaciones, inasistencias, apoyos y observaciones de cada alumno, despues de lo cual puede generar un archivo que se carga en el sistema de control escolar.

    1.-¿Cómo se genera un respaldo?

    • Una opción es:

    1. Semanalmente, al entrar al Sistema, aparece la pregunta de que si desea hacer respaldo, para ello se elige la opción Sí.

    2. Al presionar el botón Sí, aparece esta pantalla:

    El respaldo se crea en C:\RESPALDOS_ADMINISTRACION_ESCOLAR\NOMBRE DE ARCHIVO

    3. Enseguida pregunta si también se desea guardar en otra unidad de almacenamiento como USB, disco, etc.

    4. Al elegir la opción Sí, se muestra la pantalla donde se elige la ubicación del medio de almacenamiento en el que se desea guardar.

    5. Y para finalizar, arroja una pantalla indicando que el archivo se guardó correctamente.

    • Otra opción dentro del Sistema es la siguiente:

    1. En el menu de Administración, se elige en las Opciones de Administración del Sistema, Mantenimiento y Respaldos del Sistema. Y se presenta esta pantalla:

    2. Se elige la opción Respaldar Base de Datos, para que ejecute el proceso

    3. Después de ejecutar el proceso se presenta esta pantalla:

    En esta opción le dice que ya esta creado el respaldo en la unidad C: y proporciona la dirección donde se creo (C:/RESPALDOS_ADMINISTRACIÓN_ESCOLAR/ESCOLAR_27_08_2009_(09_46)) y pregunta si desea crearlo también en USB u otro medio.

    Como la opción lo índica esta opción Respalda las Bases de Datos, por lo que, para poder accesar a la información de éste, se necesita que la computadora, tenga instalado el Sistema de Administración Escolar, y se cargar la información.(Véase Cargar información de una Escuela).

    <<REGRESAR>>

    2.-¿Cómo cargar información (respaldo) de una escuela al Sistema de Administración Escolar?

    • En el menu, Escuelas, y en el submenu Borrar y Cargar Información de Escuelas, se elige Cargar información de una escuela.

    • Se presenta la siguiente pantalla:
    • A continuación pide la dirección donde está el respaldo de la escuela a cargar.
    • Una vez especificado el respaldo a cargar, se elige Aceptar, y presenta esta pantalla.
    • Se selecciona la escuela a cargar con un clic.
    • Se presenta esta pantalla que indica la escuela a cargar, y se presiona Continuar, o Cancelar en el caso de que el respaldo a cargar no sea el deseado.
    • Se presenta esta pantalla de confirmación y se presiona Sí.
    • Y por último se presenta la pantalla que indica que el proceso ha sido terminado.

    NOTA: en el caso de las Zonas Escolares, este procedimiento es necesario repetirlo por cada escuela.

    <<REGRESAR>>

    3.-¿Como se reinstala el Sistema, en una computadora donde ya lo tenía, pero que me marcó error en el sistema?

    Al instalar nuevamente en una computadora donde ya había estado instalado el Sistema, necesitamos crear una copia de la carpeta ADMINISTRACION_ESCOLAR (ubicada en la unidad C), cambiándole el nombre (ciclo anterior) y enseguida borrar de la unidad C las carpetas:

    • ADMINISTRACION_ESCOLAR
    • GENERADOR_RESPALDOS_SAE
    • RESPALDOS_ADMINISTRACION_ESCOLAR
    • CONSULTAS (existe cuando se generó alguna consulta en el Sistema)

    Una vez eliminadas todas las carpetas mencionadas anteriormente, se procede a Instalar nuevamente.
